Welcome to the PADME

Playground Main screen

This is the main screen of the PADME Playground. It splits in two components: The editor on the left and the info view on the right-hand side.

You can develop the analysis task and its components in the editor. It provides syntax highlighting, search, and other features to make your life easier. Moreover, you can start a simulation run with your current code using the play button.

The info view provides you with information about the data sources, the log output of a simulation run, files and folders changed during the task simulation, and settings for the environment variables. Moreover, you can change the route (i.e., execution order) of the data sources during the simulation.

Please visit the documentation via the link in the top right corner to get further details about each component and its capabilities.
