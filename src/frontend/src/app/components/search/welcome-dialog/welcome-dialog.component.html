Welcome to the PADME Playground! This application lets you execute analysis tasks in a simulated PADME ecosystem. With this simulation, you can test your analysis task before distributing it to data holders.

In a first step, please select the stations and related data sources you want to use for your simulation.
For each station, at least one data source is required, but you can also choose multiple if needed.

After this first step, you can upload or create the analysis task and start the simulation in the next screen.
